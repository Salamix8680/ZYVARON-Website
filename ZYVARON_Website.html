FAQ

Common questions

Is ZYVARON free?
+
Yes. ZYVARON is free to download and use. We plan to offer a Pro tier in the future with multi-device management and advanced threat intelligence, but the core agent and dashboard will always remain free.
Does it slow down my computer?
+
No. ZYVARON is designed to be lightweight. The agent runs in the background using minimal CPU (typically under 2% during scans). Between scans it uses essentially zero resources.
Do I need to know cybersecurity?
+
Not at all. ZYVARON is designed so that anyone — from a student to a CEO — can be protected without touching a single setting. Install it and it works. The dashboard is there if you want to see what it's doing, but it handles everything automatically.
Does it replace my antivirus?
+
ZYVARON complements your antivirus — it doesn't replace it. Traditional antivirus detects known malware. ZYVARON blocks attack vectors (open ports), monitors file integrity, and finds unpatched software vulnerabilities. Together they provide much stronger protection.
What data does ZYVARON collect?
+
ZYVARON runs entirely on your local machine. No data is sent to any external server. The only external connection is to the NIST NVD API to check for CVEs — this is a read-only, public API. Your files and system data never leave your device.
How is ZYVARON different from Windows Defender?
+
Windows Defender focuses on malware detection. ZYVARON focuses on attack surface reduction — blocking open ports before attackers reach them, monitoring file integrity in real time, scanning for software vulnerabilities, and recovering files after any deletion. They solve different problems.
